## Visual Reference Card Decision

Exact reusable photos for the listed NSIC rice varieties were not verified. PinoyRice provides exact variety data pages, but not reusable exact variety photos. PALAYSCAN therefore does not show generic rice-field photos on variety cards. Variety entries use visual reference cards built from maturity, grain type, yield, and environment data until the team can add own photos or confirmed licensed variety photos.

Some pest and disease entries also do not have verified reusable exact photos yet. Those entries use visual reference cards built from the local scientific name, symptoms, and identification notes. This keeps every Guide entry visually complete without pretending a generic or generated image is a confirmed field photo.

## Adding More Images Later

- Use own/team field photos first when possible.
- Use Wikimedia Commons only when the exact file page has a clear reusable license.
- Use Bugwood images only when the detail page has a reusable license suitable for the app.
- Record the source page, license type, license URL, author/attribution note, and local filename before wiring the image into app data.
- Avoid adding variety photos unless the photo actually matches the named variety or is clearly labeled as a generic rice plant/field reference.
- Keep images local in `assets/guide/` so the Guide remains available offline.
