# Allowed vs Not Allowed Image Sourcing Cheat Sheet

## Allowed

- own rice field photo
- own non-rice plant photo
- project-team photo captured specifically for the dataset
- Wikimedia Commons image with a clear reusable license
- explicitly licensed public dataset image
- academic or agriculture image with clear written permission or explicit reuse rights

## Not allowed

- random Google Images result with no license check
- Facebook image without permission
- website image with no reuse rights stated
- private chat image without consent
- social media image copied without permission
- “looks public” image with unclear ownership

## Quick rule

If you cannot clearly answer:

- where did this image come from?
- am I allowed to reuse it?
- do I need attribution?

then do not add it to the dataset.
